# Calculadora de Média de Aluno

Este projeto foi desenvolvido no início do semestre da faculdade como parte do curso de Análise e Desenvolvimento de Sistemas (ADS).

##  Descrição
O programa calcula a média de um aluno com base em duas notas e verifica se ele foi aprovado ou reprovado, considerando também a frequência.

##  Como funciona
- O usuário informa:
  - Nota 1
  - Nota 2
  - Frequência
- O programa calcula a média das notas
- Critérios para aprovação:
  - Média maior ou igual a 6
  - Frequência maior ou igual a 75%

## Tecnologias utilizadas
- Linguagem C

## Como executar
1. Compile o código com um compilador C

2. ## Aprendizados
- Entrada e saída de dados em C
- Uso de variáveis
- Estruturas condicionais (if/else)
- Operadores lógicos (&&)

Este projeto foi criado com fins educacionais e representa um dos meus primeiros contatos com lógica de programação.
